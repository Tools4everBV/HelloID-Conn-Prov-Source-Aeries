# Setup Aeries API Set up

- Click Add to create a record for a 3rd party product.
- Enter "HelloID" for Product Name, Status should be Active
- Take note of the certificate key
- Setup permissions
  - Student Data
    - Student Data
  - Scheduling
    - Teacher Data
    - Master Schedule
    - Course Data
    - Classes
  - School Information
    - Schools
    - Terms
    - Users
